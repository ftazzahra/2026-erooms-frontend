# Changelog

All notable changes to this project will be documented in this file.

This project follows Semantic Versioning.

---

## [1.0.0] - 2026-02-17

### Added

- Initial frontend setup using React + TypeScript + Vite – Project initialized using Vite with React and TypeScript configuration.
- Landing Page – Public landing page displaying general information about the room booking system.
- Authentication Pages (Login & Register) – Implemented user authentication interface for account login and registration.
- Admin Dashboard – Dashboard interface for administrators to monitor and manage system data.
- User Dashboard – Dashboard interface for regular users to access booking features.
- Profile Page Feature – Page allowing users to view and manage their personal information.
- Admin Rooms Management – Feature for admin to create, update, and manage room data.
- Admin Bookings Management – Functionality for admin to view and control room reservations.
- User Rooms Page – Page for users to browse available rooms.
- User Booking History – Feature allowing users to view their past booking history.

### Changed

- Refactored Layout Structure (Navbar, Sidebar, DashboardLayout) – Improved component organization and UI consistency.
- Updated Routing Configuration (App.tsx) – Adjusted routing to support role-based navigation and profile integration.
- Improved UI Consistency – Unified layout and navigation flow between admin and user dashboards.
